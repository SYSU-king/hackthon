## 2. 项目概述
**项目名称：** 
LifePath Engine（生命轨迹推演仿真引擎）

**项目简介：** 
LifePath Engine 是一款基于大语言模型（LLM）驱动的多智能体生命轨迹仿真引擎。本项目致力于解决社会政策制定、教育资源分配以及个人职业规划发展等场景中缺乏长远反馈的问题。实现思路上，我们利用 FastAPI 构建后端逻辑，通过 LLM 赋予数字环境中的“智能体”内在动力学与认知演化能力，在虚拟沙盘中推演不同社会干预措施（如教育政策、福利制度）对个体一生的长远影响，并在前端进行可视化呈现。项目不仅具备技术探索价值，也为社会科学研究提供了低成本的辅助决策实验场。

## 3. 创新点
**你的项目有哪些独特或创新之处？** 
1. **LLM原生驱动的多智能体仿真**：有别于传统基于既定规则的仿真沙盒，本项目利用大语言模型驱动角色内心的认知变化、决策逻辑与成长路径，使仿真结果更贴近真实存在的不确定性与普遍人性。
2. **聚焦社会价值量化的“AI for Social Good”**：能够具象化评估如贫困帮扶、教育资源倾斜等社会政策的长期红利，让“社会良善”与“可持续性”的影响变得可衡量、可预见。
3. **宏微观结合的沙盘架构**：不仅能够追踪单个智能体（Micro层面）在各种岔路口的命运轨迹，更能汇总生成宏观（Macro层面）的社会群体演变报告，多维度展示技术向善的潜力。
